# Flutter web 工程目录初始化
适用于 linux | macos

# 使用
#### curl
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/eavesmy/flutter_new_web/master/install)"
```

#### wget
```
sh -c "$(wget -O- https://raw.githubusercontent.com/eavesmy/flutter_new_web/master/install)"
```

#### 直接使用
```
./install [project name]
```
